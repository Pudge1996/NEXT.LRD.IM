> 因为以前选课忽略了一些必要的规则，导致我大四仍然在学校上一些选修课，错过了一些挺好的实习岗位。在此期间要保持学习，利用大学最后一个学期（03/2018~06/2018）提升自己的能力。

回归正题，大四下学期的计划：

## 一、毕业设计改版

在寒假前我已经基本完成毕业设计的作品了，回到广州之后让朋友使用了一会，也给了我一些意见。自己整理了一些待修改位置，回到学校要抓紧时间修改，做一个《刁角武汉 V2.0》。同时修改后的文档，包括最后的设计文档也会也会发上 Medium。

可选计划：做一个介绍《刁角武汉》的 MG 动画。我对 MG 动画没有深入的了解，如果要做毕设的 MG 动画应该要花不少时间，所以列为可选计划。

## 二、完善个人网站 Pudge1996.com

学习永无止境，我一直认为做互联网方向设计师要拓宽自己的视野和知识面。前端知识也是 UI 设计师有必要了解的方向。在这个学期，我会完善我的个人网站，包括：

1. 添加视频展示。寒假前试过用腾讯云的视频点播服务，但是因为个人技术上的原因展示并不完美，便搁置了这个计划。这个学期要解决这个问题；
2. 优化使用体验。比如点击图片放大，调整超链接 Hover 状态等；
3. 使用懒加载 JS。之前试过一轮发现没效又放弃了。
4. 更换字体。目前我对网站上的字体并不满意，原因是字体太细，轻飘飘的没有重量感，其实看起来有点略微费劲。最好是换一个中文和英文都看起来舒服，正常的字体。

## 三、花更多时间学习 UI 设计

在暑假的飞滴项目之后我已经很少接触 Sketch 了，最近的作品多数用 Illustrator 完成，这个学期我要花更多时间学习 UI 设计，熟练利用 Sketch 的 Symbol 做一些 UI 组件。

同时要参加 UICN、站酷的一些 UI 设计比赛，以及 [Sketch Together](https://www.facebook.com/groups/sketchtogether/) 和 [Uplabs](https://www.uplabs.com/challenges) 的 Challenges。

## 四、接触一些运营设计

在寒假面试的公司都有做运营设计的需要，我一直对这方面经验比较少，这个学期要稍微接触一下，做一点小练习。同时运营设计偶尔会用到 C4D，我看看能否学习一些吧，3D 一直是我的弱项。

02/03/2018 李瑞东 在江汉大学，武汉市

## 1. 更换网站字体 07/03/2018


删掉了原来瓜皮有字库的字体引用，只修改 CSS 即可达到不错的字体效果。

```css
body {
    font-family: “Helvetica Neue”, Helvetica, Arial, sans-serif;
}
```

添加了 CreativeMarket 的超链接在 Footer 之后发现在手机上会另起一行，于是设置了在 420px 的宽度下 Footer 字体改为 13px。
